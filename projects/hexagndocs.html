Basic things

About Hexagn

Hexagn is a simple programming language that compiles to URCL. This make Hexagn very portable, since lots of transpilers and emulaters for URCL is made already.

You can even run Hexagn programs in Minecraft CPUs if it has enough bits, memory and etc.

File Extention

filename.hxgn

Compilers

There is a C++ one and Rust one. C++ one is the original compiler but lacks some features that Rust compiler has.

This document is mainly targetted the C++ one.

Syntax

Comments:

// Hello single-line comment

/*
Hello multi-line comment
*/

Call a function:

a_func_to_be_called(/*args*/);

Variable init:

var_type var_name_not_init;
var_type var_name = var_value;

Set variable:

var_name = new_value;

Math:

// Basically try to code in C and learn from it

Importing:

import name;
import name.math;

Directly writing URCL:

//urcl "ADD R0 R1 R1";
//urcl "SUB R0 R1 R2";

Conditionals:

if (/*condition*/) {
    // ...
}	// Does it have else? It doesn't seems like having one

Loops:

while (/*condition*/) {
    // Do something in here
}

Declaring functions:

return_type func_name(argN_type argN_name /*, ...*/) {
    // ...
    // optional return (?)
}

Hexagn stdlibs

io

Printing:

void println(string); // Print + a \n
void print(string);   // Print to console

Input:

void input(string); // Replaces your string with user inputted one
void input(int32);  // Basically useless

io.graphics

Set Pixel:

void setpixel(x, y, color); // Color maybe assumes target machine is using RGB24 or RGB32?

Get Pixel:

// Personal advice: Use framebuffers instead, they are faster and the only choice of some machines
int32 getpixel(x, y); // Returns the color of pixel in X, Y

Buffer:

void buffer(state); // Same as OUT %BUFFER state in URCL

io.graphics.advanced

Draw line:

void drawLine(x1, y1, x2, y2); // Draws line from (x1, y1) to (x2, y2)

io.mouse

Get mouse position:

int32 getmousex(void);
int32 getmousey(void);
// Same as IN RX %MOUSE_X and %MOUSE_Y in URCL

Get mouse button state:

int32 getmousebuttons(void); // Same as IN RX %MOUSE_BUTTONS in URCL

util

It contains no function.

util.math

Absolute:

intX abs(intX);  // Same as ABS in URCL

Sign:

int8 sign(int16);
/*  Returns  0 if arg = 0
    Returns  1 if arg > 0
    Returns -1 if arg < 0
*/
